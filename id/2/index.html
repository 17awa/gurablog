# 舍弃自卑的我

从前的我，我觉得我哪里都不如别人，走在哪里觉有人在看我，很不自信，

每天都一笑而过，感觉很不自在，这种感觉太难受了！

可是现在的我，每天大大方方的走路，你以为别人在看你其实根本没有人会注意你，

我每天穿什么穿什么，还能去冰李游泳根本没人管,怎么有一股味道，

这味道从哪里发出来的?原来是从我身上发出自信！

中国有句话叫作自信既巅峰，不是写错,是自信既顶峰，有一次我在外面迷了路，

我直接叫停一辆车，我直接上车，让司机送我一下，只有口才好，到哪都无敌，

夸人家两句就到家了，还有一次是我在外面找不到妈，我直接找

到工作人员开喇叭，然后我去人群中不喊，根本没有人理我，这就是自信的味道，就像这篇作文一样，

因为我自信才能写出来这种感觉有病的作文。

那对于我自信这件事呢，让我说两句，那就说几句，首先啊，这几句要比那几句强得多，

为什么这么说呢，因为啊，这句也比那几句强，所以我要说哪几句呢就说这几句，这几句啊，

可比那几强得太多，为什么这么说呢，啊，你们也知道，是什么呢，啊，是比那几句要太

强得多了，所以就这几句来讲，嘿，您猜怎么着,没了。

最后，我从生中走了出来，相信我，只要有自信，就可像我这写出自信的作文。

我发现我也是一个自信的人
