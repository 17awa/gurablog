# 记忆深处

今天我突然想起我在外面迷了路，被一个好心人救了，

他把我送回家，还请我吃糖，他人还怪好的。

那天，我出去玩，在外面迷路了，我走呀走呀走呀走，走了好久，

还在原地，我真的服了，我只能一真走，终于碰到了那个好心人，

他问我是不是迷路了，我说是，他说要送我回家，我说行，他让我上车，

我上了车，他让我吃糖，我吃了糖，他说让我喝水，我喝了水，他把我送到了家，

问我用不用开上楼，我说不用了，走之前，我感谢了这个好心人。

现在我想找到这个好心人感谢他，这个人太好了，没见过这么好的人，太好了这个

好心人。我也要当一个好人，可以帮助别人，可以积德，还可以立一个好名声，总之，

我要当一个好人，用自己的善良让更多人像我一样以后当一个好人。

这件事情让我很难忘，我忘不了这个好心人，大家也要做一个好人去帮助别人，

不要当坏人，这个好人一直在我记忆深处。
