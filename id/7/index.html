# 路上趣事

今天，我要去方山县游玩，今天早上我早早的起了床

因为我太高兴了。我们出发后，在路上遇到了一

个穿拖鞋骑电动车的大妈，听说我们要去方山县玩，

她说她也是方山县的，给我们说了几个好玩的地方有方山动物园，

方山锦汤，是一个叫锦师父做的汤，闻起来奇臭无比，喝起来也奇臭无比，

我们说了几句就走了。走到路上，看到一个人把冰块放到了裤档里，笑死人了，

还看到一个在外面写作文的大师，我们下去看了一下，真是写的一手好字，

我没见过这么好看的字。终于到方山了，到了之后，先给家人报信：“到达世界最高县，方山县，

看看远处的锦汤饭店，看，那不是方山动物园吗？”我们先去方山动物园看了动物，

看到了世界上最高的龙的化石，真的太好看了，长得真高，实在是高，导游告近我们这个龙的

名字简称龙高，我们觉得也很合理……最后，我们玩了很多，怀着开心的心情回去了，

但我还是要感谢那穿拖鞋骑电动的大妈，最后我才知道她就是锦汤创史人，名锦姓木子。
