# 大妈

放学了，我急匆匆地赶回家，因为我很急，为什么急，

因为我妈让我赶快回去，为什么能回去，因为放学了。

我在路边买了点奥润鸡就急匆匆往家赶，到了家，我看到

家里有一辆黑色电动车，沙发上有一位穿着绿色托鞋的大妈在那，

旁边有只黑色杜宾犬，嘴里有一个不知道是谁的作文本。

我叫小狗去玩，它却从我的柜子里面变魔术，一会拿出来一盆花，

太神奇了，可是他怎么看起来快气死了我给了大妈一个奥润鸡他说好吃

可她看着像面部扭曲了，我问也怎么了，她说小脑萎缩了。

最后，大妈穿着经托鞋骑着黑色电动车走了，杜宾犬在座上，今天的大妈和杜宾大可真奇怪，
