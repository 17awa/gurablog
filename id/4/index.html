# 快乐的春节

今天过年啦！我太开心了，今天我早早的就起床了

因为太开心了，爸爸妈妈在炸年糕。

我们一家去拜年，到了我表哥家收了压

岁钱，又去了表弟家；又收了压岁线，又去了堂哥家，

还收了压岁钱，还去了堂羊家又收了压岁线。

中午 戚来我家吃饭，今天的饭菜都

很香，还没吃就想吃了，让人吃了还想吃今天有，鱼、鸡、鸭、鹅，猪……

吃完饭我就和好朋友小龙去打雪仗，他打我一下，我一下，打了半天，还在打，

最后我直接一个天旋地转丸把他打倒，

他也起来一个超级无敌世界最强牛逼世上无敌超超级级

世上最强无敌超级牛逼别炸天秒天.秒地、秒空气加宇宙超级宙斯

无敌罗旋超强地爆丸把我Ko。

晚上，我们在一起看小品，小品里面说的北京话，听起来郎郎上口，

既好笑又好看由其是到了搞笑片段，我们笑的哈哈的，还有一句话是：“您猜怎么着，没了”

我太喜欢过年了，我从小品学了一句话嘿，您猜怎么着，没了才怪才怪。
