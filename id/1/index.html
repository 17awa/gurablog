# 毕业致词

敬爱的老师，你好，我好想不离开这里，

这里在我们的回犯，我们一起走过6000年多个日日夜夜，

一共六年，过36个盛夏6个寒冬

在毕业之前，和建学校不要住我们班招一些不三不四的东西，

太臭了，还有建议把学校的安全做好，不要让学生有伤

记得有一次，在学校玩面时，不小心摔倒了老师看到了过来把我扶了起来，

还有一次是同学们看到我一个人在那玩，就叫我和他们一越玩，我还认识了我们班的大明星高家好，

他在我们珍是出名，就连附近的辅导班都认识他他的是方山县的明星，在这里也是大明星，我们都是他们的粉丝，我好想看看大明星是什么样。

还有我的老师，他每天二课下课，每天就是这样，这事你说哎，这事也不好，嘿他每天哎呀去上课，我们也是也你说这怎么好拿，你说也是吧！啊，还有你说呢，你觉得这事怎么样，哎，这事哎呀你这是吧，那让我说几句，那

我就说几句吧。最后，我祝学校、老师、同学：寿比南山、福如东海，永远不死，万事如意，一笑顷城，家财万贯，早出壮元左手，又手大，年年有余，步步高升，犹如仙女，发发财财，身体连康，事事如意，活过阎王，一笑而过，

掩耳倒邻守株待象，亡牢补狼，斋古开山，我常鹅造人，石头出猴，齐天大车，大午天宫，拔高龙锦，八仙过海，牛郎织女，最后祝大家：好好学习，天天向上。
