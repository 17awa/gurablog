# 其实,我真的想不写这篇作文

其实，我真的想不写这篇作文，

因为我不想写。为什么不想写，因为我太懒了，为什么太懒了，因为出去玩太累了。

我真的不想写这篇作文，因为其他作业太多了，我真的不想写，才休息一天，还有那么多作业，连门都出不了，

我想出去玩，可是作业还没有写完，我真的不想写这篇作文。

我要去上补课班，上午一节课，下午两节课，每天要8点才能回家，我吃完饭就开始写作业，要先把学校的写完，

再写补习班的作业，这篇作文还是我都写完作业后才想起来的，写完己是深夜两点。

当我正准备睡觉时，才想起来还要复习，我又爬起来，开始复习，一直到五点才睡，第二天，

七点就又爬起来上学，我真不想写这篇文。

我感到现在的学生压力山大，每天都要写这么多的作业，还要每天上补习班，真的太难了，

还要复习，唉真的太难了，每天作业不到三四点别想睡觉，我真的太难了，太难了。

最后，我真的想不上写这篇作文！
